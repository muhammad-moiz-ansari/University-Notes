Lecture 01: Course Introduction & Trends

  1.1.3. Moore’s Law vs. actual performance gains

  1.2.1. What defines an HPC system?
  1.2.2. Latency-oriented (CPU) vs. Throughput-oriented (GPU) design

  2.1.1. Memory Hierarchy (Registers -> L1/L2 -> RAM -> Disk)

Lecture 05: CUDA Programming Basics (Part 2)

  8.2.2. Pinned Memory (Page-locked) vs. Pageable Memory

Lecture 10: Advanced Memory Optimizations

  10.2.2. Tiled Algorithms (Stencil computations)

  11.2.1. Hiding data transfer costs (Copy/Compute overlap)
  11.2.2. Stream Synchronization (cudaStreamSynchronize)

Lecture 12: OpenACC

  12.1.1. Syntax: #pragma acc
  12.1.2. kernels vs. parallel regions
